# My Web Projects Collection

This is a collection of web projects including a Weather Forecast app, a Simple Calculator, and a Number Guessing Game.

## Projects Overview

### Weather Forecast App

A React-based weather application that allows users to check weather conditions for different locations.

**Technology Stack:**
- React
- TypeScript
- Weather API

**How to Run:**
1. Navigate to the weather website directory: `cd "weather website"`
2. Install dependencies (first time only): `npm install`
3. Start the development server: `npm start`
4. Open your browser and go to: http://localhost:3000

### Simple Calculator

A basic calculator with a clean UI that can perform arithmetic operations.

**Technology Stack:**
- HTML
- CSS
- JavaScript

**How to Run:**
Simply open the `simple calculater/index.html` file in your browser.

### Number Guessing Game

A fun game where you try to guess a random number between 1 and 100.

**Technology Stack:**
- HTML
- CSS
- JavaScript

**How to Run:**
Simply open the `number gussing wensite/game.html` file in your browser.
